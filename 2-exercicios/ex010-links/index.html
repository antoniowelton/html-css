Usanso Links Externos

Podemos criar um link usando a tag <a> e </a>

Depois de abrir a tag, usamos os parâmetros href = "" para colocar o caminho do link completo. Em seguida, temos o parâmetro TARGET = "_blank" que abrirar o link em outra aba.

Também tem o parâmetro REL = "external" que serve para melhorar os resultados nas buscas do Google.

Sintaxe da TAG a para links

        
            <a   href="caminho.completo.do.link."   target="_blank"   rel="external"> frase ou palavra que será clicável </a>

Alguns exemplos de links

  1. Esse é meu Canal no YouTube

  2. Esse é meu Instagram

  3. Esse é o meu Facebook

  4. Esse é o meu LinkedIn
